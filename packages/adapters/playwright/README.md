# @page-component-object/adapter-playwright

> **Start here:** [Page Component Object (PCO) — main README](https://github.com/dvegap95/page-component-test-object#readme)

Experimental Playwright adapter comparing Locator + await model vs Cypress chains. **Do I need this?** Not for production — research only; both E2E paths stay open.
